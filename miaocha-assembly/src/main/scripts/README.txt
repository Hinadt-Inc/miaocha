==========================
日志管理系统 - 快速指南
==========================

这个目录包含了管理日志系统的所有脚本，所有的脚本都有彩色输出和用户友好的交互。

可用脚本:
---------

1. 启动应用
   - start.sh / start.bat
   
   选项:
   --env [环境]    指定运行环境 (dev, prod)
   --debug        启用调试模式
   --help         显示帮助信息

2. 停止应用
   - stop.sh / stop.bat
   
   选项:
   --force        强制停止应用
   --help         显示帮助信息

3. 重启应用
   - restart.sh / restart.bat
   
   选项:
   与启动脚本相同，并增加了 --force 选项

4. 查看状态
   - status.sh / status.bat
   
   显示应用的详细状态信息，包括:
   - 运行状态和进程ID
   - 内存使用情况
   - 端口监听状态
   - 运行时间
   - 当前活动环境

示例用法:
--------

启动生产环境:
./start.sh --env prod

优雅停止应用:
./stop.sh

强制停止应用:
./stop.sh --force

查看应用状态:
./status.sh

更多详细信息，请参考系统根目录下的README.md文件。
